Update Instructionsâ€‹

This history file should be automatically updated whenever new content is generated for the textbook. All new modules, chapters, code examples, diagrams, and notes should be documented here in chronological order.

Automatic Update Systemâ€‹

The textbook history is maintained using automated scripts:

  • update-textbook-history.sh (Bash script for Linux/Mac environments)
  • Update-TextbookHistory.ps1 (PowerShell script for Windows environments)

These scripts can:

  • Scan the docs/ directory for new content
  • Automatically add new files to this history document
  • Extract module and chapter information from file names and content
  • Maintain chronological order of content additions

To use the automatic update system:

  • On Windows: Run Update-TextbookHistory.ps1 -Scan to detect and add new content
  • On Linux/Mac: Run ./update-textbook-history.sh --scan to detect and add new content
  • Or call the script directly with specific parameters when creating new content
